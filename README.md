# Ramadan Greeting Card - Golden Luxury Edition

## Overview
An elegant web application for creating premium golden Ramadan greeting cards with professional design. Users can customize cards with names and photos, then download and share them.

**Live Demo:** https://mragab22.github.io/-ramadan/

## Key Features

**Design**
- Luxury golden theme with gradient backgrounds
- Animated decorative elements (lanterns, crescents, stars)
- Fully responsive for all devices
- RTL Arabic interface

**Functionality**
- Two modes: gift for others or personal card
- Custom name input for sender and recipient
- Optional photo upload
- Live preview
- Typewriter text animation
- Golden confetti celebration effects
- High-quality PNG download (3x scale)
- Print-optimized output

## Technologies

**External Libraries**
- Canvas Confetti for celebration effects
- html2canvas for image export

**Fonts**
- Tajawal, Amiri, Almarai (Google Fonts)

**CSS Features**
- CSS Variables
- Animations and Keyframes
- Flexbox layouts
- Responsive clamp() functions

## Browser Support
Chrome, Firefox, Safari, Edge - all modern browsers supported

## Usage
1. Open the HTML file in any browser
2. Choose mode (gift or personal)
3. Enter names and optional photo
4. View animated card with confetti
5. Download or print

## File Structure
- Single HTML file containing all code
- No server required
- Works offline
- All processing done locally in browser

## Technical Highlights
- No backend needed
- Fast loading
- Secure local processing
- Clean, documented code
- 3x scale for high-quality export
- Mobile-optimized with special breakpoints
